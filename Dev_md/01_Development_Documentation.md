# 1. Rokey News Project 개발 문서

## 1.1. 프로젝트 개요
- **프로젝트명**: Rokey News Project
- **목표**: 최신 뉴스를 효과적으로 사용자에게 전달하는 웹 애플리케이션 개발.
- **주요 기능**:
  - 뉴스 기사 목록 표시
  - 프로젝트 소개(`README.md`) 내용 렌더링

## 1.2. 기술 스택
- **프론트엔드**: React, TypeScript, Vite
- **스타일링**: CSS
- **버전 관리**: Git, GitHub

## 1.3. 향후 개발 방향
- 뉴스 카테고리 기능 추가
- 키워드 검색 기능 구현
- 사용자 맞춤형 뉴스 추천 기능 도입
