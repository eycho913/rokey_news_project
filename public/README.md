# Rokey News Project

**Rokey News Project**는 최신 뉴스를 효과적으로 사용자에게 전달하는 것을 목표로 하는 웹 애플리케이션입니다.
이 프로젝트는 React, TypeScript, Vite를 사용하여 구축되었습니다.

## 🚀 주요 기능

- **실시간 뉴스 피드**: 주요 뉴스 기사 목록을 실시간으로 제공합니다.
- **반응형 UI**: 데스크톱, 태블릿, 모바일 등 다양한 디바이스에서 최적화된 화면을 제공합니다.
- **프로젝트 문서 조회**: 웹 애플리케이션 내에서 이 `README.md` 파일의 내용을 직접 확인할 수 있습니다.

## 🛠️ 기술 스택

- **프론트엔드**: React, TypeScript, Vite
- **상태 관리**: React Hooks
- **스타일링**: CSS Modules
- **버전 관리**: Git & GitHub

## 📄 개발 문서

프로젝트의 상세한 개발 가이드, 규칙, 평가 기준 등은 `Dev_md` 폴더에 정리되어 있습니다.

- `Dev_md/01_Development_Documentation.md`
- `Dev_md/02_Rules_and_Guidelines.md`
- `Dev_md/03_Content_Evaluation.md`

## 🏁 시작하기

1.  **리포지토리 클론**:
    ```bash
    git clone https://github.com/eycho913/rokey_news_project.git
    cd rokey_news_project
    ```

2.  **의존성 설치**:
    ```bash
    npm install
    ```

3.  **개발 서버 실행**:
    ```bash
    npm run dev
    ```

4.  **빌드**:
    ```bash
    npm run build
    ```

이 프로젝트는 학습 및 포트폴리오 목적으로 제작되었습니다.
